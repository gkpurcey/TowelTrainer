Towel Trainer Configuration Screen

Rotation Rate (Hz)

Frequency used to pulse the towel rotation motors (400 steps per full rotation)

Rotation Headstart (ms)

Time in ms to allow the towel to spin before starting the tape sled

Sled Rate (Hz)

Frequency used to pulse the sled motor (200 steps per full rotation)

Sled Passes (ms)

Number of passes for the tape sled to travel between the two end stops

Sled Endstop Pause (ms)

Time in ms to allow the towel to spin at the endstop between each sled pass
